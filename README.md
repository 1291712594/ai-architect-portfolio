**产品经理：**朱东刚

## 📖 项目简介

**智能建筑规范助手**是一款基于RAG（检索增强生成）技术的AI智能问答工具，专为建筑行业从业者设计。它通过自然语言交互，帮助建筑师、工程师和学生快速、准确地查询建筑规范条文，并提供深度解读、案例参考和关联推荐，旨在成为建筑从业者的“规范查询神器”。

> **核心理念**：用AI技术解决建筑师在规范查询中的痛点——繁琐、难懂、易遗漏、更新滞后。
> 

## ✨ 核心特性

- **自然语言问答**：像聊天一样提问，无需关键词检索，直接获得精准答案。
- **条文溯源**：每条回答都附带规范原文出处（如“GB50016-2021 第X条”），点击可查看原文。
- **深度解读**：提供条文的通俗解释和典型设计案例，帮助理解规范背后的逻辑。
- **关联推荐**：根据当前问题智能推荐相关条文，避免遗漏关键规范。
- **规范更新提醒**：自动监控规范变更，及时推送更新通知。
- **社区贡献**：资深设计师可提交自己的案例和解读，共建知识库（迭代中）。

## 🎯 目标用户

- **资深建筑设计师**：快速确认方案合规性，获取条文权威解读。
- **青年工程师/助理**：高效学习规范，避免理解偏差。
- **建筑系学生**：轻松入门规范知识，辅助课程设计。

## 🛠️ 技术栈

- **前端**：HTML/CSS/JavaScript（MVP阶段为简易Demo）
- **后端框架**：[Dify.ai](http://dify.ai/) 开源版（RAG框架）
- **向量数据库**：Chroma（轻量级）
- **AI模型**：OpenAI API / Deepseek API（可根据需求切换）
- **部署**：支持私有化部署（企业版）

## 📚 使用指南

1. **提问**：在输入框中用自然语言描述你的问题，例如“18层住宅的楼梯间要封闭吗？”
2. **查看答案**：系统返回规范条文、解读和案例，底部附有原文来源。
3. **溯源验证**：点击来源链接可展开规范原文截图，确保答案可靠。
4. **探索关联**：右侧推荐相关常见问题，点击快速跳转。
5. **反馈**：对答案点赞/点踩，帮助我们改进。

## 🧪 MVP验证指标

- **查询速度**：平均单次查询时间 < 5秒。
- **答案准确率**：针对高频问题，准确率 > 90%。
- **用户满意度**：点赞率 > 80%。
- **留存率**：测试用户7天留存率 > 30%。

## 🗺️ 迭代路线图

- **阶段一（MVP，1-2个月）**：三本规范知识库，基础问答+溯源，Web Demo。
- **阶段二（社区化，3-4个月）**：扩展至10本规范，增加社区贡献、计算工具（疏散宽度计算）。
- **阶段三（商业化，6个月+）**：企业版私有化部署，AutoCAD/Revit插件，20+规范覆盖。

## 🤝 如何贡献

我们欢迎社区贡献！无论是提交bug、建议新功能，还是分享你的规范案例，都可以通过以下方式参与：

1. **提交Issue**：在GitHub仓库中创建issue，描述你的想法或问题。
2. **补充案例**：在MVP上线后，可通过“补充案例”按钮提交你的实际项目案例，经审核后展示给所有用户。
3. **代码贡献**：fork仓库，提交PR（请遵循现有代码风格）。

## ⚠️ 风险与应对

- **技术风险**：初期使用低成本API，建立用户反馈机制优化模型。
- **市场风险**：通过免费试用和对比教程吸引用户，突出差异化优势。
- **内容风险**：专人审核规范更新和社区内容，确保准确性。
